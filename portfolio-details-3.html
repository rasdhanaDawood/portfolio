    1. E-commerce Platform: Users can browse, select, and order cupcakes online.

    4. Frontend Design: Clean and responsive UI built with HTML and CSS for a user-friendly experience.
    5. Backend Development: Built with Node.js and Express.js, ensuring scalable and fast server-side operations.
    6. Admin Panel: Includes a dedicated admin interface for managing:

      • Products (add, update, or delete products)
      • Orders (view and update order status)
      • Users (manage customer details)
      • Sales analytics and reports
